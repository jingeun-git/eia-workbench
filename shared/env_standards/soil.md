# 토양오염우려기준·대책기준 DB — `soil.json` 요약

> 원본: [[soil.json|soil.json]] — 워크벤치 `envdata.js`가 읽는 실제 데이터(이 문서는 인덱스 요약본)

- **근거법령**: 토양환경보전법 시행규칙 별표3(토양오염우려기준, 제1조의5 관련)·별표7(토양오염대책기준,
  제20조 관련), 개정 2025-08-07
- **검증 방법**: `law_client.py annex --mst 287443 --no 0003(우려기준), --no 0007(대책기준)` 원문
  직접 조회(2026-07-22)
- **구조**: `type: "region", columnsFixed: false` — 소음·진동(고정 2컬럼)과 달리 **항목이 대기질처럼
  추가/삭제 가능**하고(23종 중 실제 측정한 것만 추가), 지역구분(1·2·3지역)은 측정지점마다 선택한다.
- **두 기준 동시 비교**: 각 항목이 `values`(우려기준)와 `actionValues`(대책기준) 두 값을 갖는다 —
  우려기준 초과는 정밀조사 대상, 대책기준 초과는 정화명령 대상으로 법적 효과가 다르다.
- 23개 항목 중 다이옥신만 단위가 다르다(pg-TEQ/g, 나머지는 mg/kg). 유기인화합물은 대책기준이
  원문에 없다("-" 표기 → null).
